Please change the domain name first

Each controller in your local network should have a unique "domain name", which identifies it on the local network. This controller is currently still using the default name. Having two controllers with the same default domain name on the same network will cause problems. Before you add a second controller, make sure to first change this controllers' hostname into something different and unique. It should be one word, such as "kitchen" or "bedroom".

After you changed the name you will then be able to access this controller by typing that new name into a browser (whenever you're on the local network). For example, it might become kitchen.local instead of candle.local.

Chat with your home

You can chat with Voco a privacy friendly and encrypted messaging network called Matrix. This is very similar to Whatsapp or Signal. There are many ways to use it, such as installing a Matrix app on your phone or tablet, or logging in through a web application.

Matrix is like a cross between email and Whatsapp/Signal. Where Apps like Whatsapp have a central server that controls the entire chat network, with Matrix it's more like email, where lots of different servers exchange the messages. And as with email you can choose what provider you trust your messages with.

This design means no single organisation owns the chat network. Anyone can create a server, and then these servers route the messages between them so they get to the intented destination. You could even run your own server if you wanted, which would give you very high levels of control over your data.

This is made possible by the Matrix software that all these servers run. Matrix is open source, managed by the Matrix.org foundation, and supported by a worldwide community of people and organisations. The open source nature means the programming code can be viewed by anyone, including you. You can even help improve it. For many people this level of transparency allows for greater levels of trust.

The Matrix software can encrypt the messages, and Voco enables this for all the messages it sends.

You can learn much more at matrix.org.

Creating a Matrix account

Candle can create a Matrix account for you, on a French Matrix server called domainepublic.net. This means your messages will be stored in France, by a non-profit foundation. Candle has no relation with this foundation, and has no control over what happens on their server. But we do trust it.

How to use Chat

Voco does only offers 100% local voice control, but you can also connect it to a messaging network called Matrix.

Matrix is very similar to messaging apps like Whatsapp and Signal (it even uses the same encryption standard), but it's even more privacy friendly. If you enable this feature, then you can literally chat with your home as if you were in a Whatsapp or Signal conversation.

Basics

Generally the reply to a command will be sent to back the where it came from. If you use your voice to give a command to Voco, then Voco will speak the response out loud. If you use a text message to give Voco a command, the response will be sent back as a text message.

There are a few exceptions to this rule:

  • Timers, alarms and reminders will always also be spoken out loud. So it you send a chat message "set a timer for 10 seconds", when the timer finishes you will receive a message saying "your timer is finished", and that message will be spoken out loud too.
  • An exception is the countdown, which will only speak out loud. This is to avoid spamming the chat room with lots of messages.

Another exception is that you can use these special text commands:

  • Start a message to Voco with speak:, and Voco will say that text out loud for you.
  • Start a message to Voco with speak everywhere:, and Voco will say that text out loud for you through all the Voco satellites as well.
  • Start a message to Voco with popup:, and the message will be shown on a popup in the interface.

Managing participants

If you use the chat functionality, then Voco will create a "chat room". You can invite multiple people to that room, and each person can send commands to Voco.

To protect the privacy of participants, newly invited members cannot see what messages were previously exchanged.

Notifications

If you create a rule you may notice that voco comes with two "notifiers". In your notifier you can set three things: a title, a message, and a priority level.

The Speech notifier will get Voco to say something out loud. You can use either the title or message fields for this message (or both). The level value currently isn't used.

The Matrix notifier can be used to send text notifications to your mobile devices. This has three levels:

  • On the low priority level the title and message are shown as a simple line of text.
  • On the medium priority level the title is shown as bold text.
  • On the high priority level both the title and message are shown as bold text. On top of that a message will pop-up in the Candle controller interface.

Enabling and disabling chat control

You can at any time enable or disable chat control of your home. The Voice Control thing has a property called "Chat". When it is off Voco will ignore any incoming commands.

You can decide if you want notifications to still be sent out when chat funcionality is disabled. Visit the Voco settings page to change your preference.

Why Matrix?

The great thing about Matrix is that it's a distributed ("federated") messaging network. This means it operates much like email, where there are multiple servers that exchange messages. You're not tied to one company, as you are with apps like Whatsapp. Anyone can run one of these servers and become part of the network.

The small downside is that this means your conversations with Voco will be stored on a third party server. We only use servers we trust. Currently this is a server by the French NGO domainepublic.net. If Candle becomes popular enough then we will setup our own Matrix server.

The Matrix software is open source, and is managed by a non-profit foundation. Learn more at matrix.org.

Privacy

Voco's voice functionality does not make any connections to the outside world. Period.

The chat funtionality will become active once you create an account or prove your existing Matrix username. Before that moment no connections to outside Matrix servers are made.

Note that disabling the chat functionality using the Voice control thing does not stop communication with the Matrix server. It's just that incoming messages are ignored, and outgoing messages are only sent if you allow this in the Voco settings.

To better understand how your data is stored on the Matrix server, please read the privacy policy of the server itself. Candle's webserver is at no point involved in the chat functionality.



Debugging is enabled

This addon is currently set to save detailed information about its operation in the internal logs. This may impact your privacy, as this also means that insights into your behaviour might be stored in those logs.
